本附录汇总后端开发中常见的构造件分类。按承接的结构分组。

E.1 Web 框架构造件

名称 语言 描述 承接的结构
FastAPI Python 异步、高性能、自动 OpenAPI API 入口结构
Flask Python 轻量、灵活、扩展丰富 API 入口结构
Django Python 大而全、ORM 内置 API 入口 + 数据访问结构
Express Node.js 最小主义、中间件模式 API 入口结构
NestJS Node.js 模块化、装饰器、DI API 入口结构
Spring Boot Java 企业级、AOP、DI 容器 API 入口 + 组合结构
Gin Go 性能极高、轻量 API 入口结构
Actix-web Rust 性能极高、类型安全 API 入口结构

class-validator Node.js 装饰器验证 输入验证结构
