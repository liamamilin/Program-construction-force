附录 D — 附录 D：前端构造件分类表（Frontend Asset Catalog）

本附录汇总前端开发中常见的构造件分类。按承接的结构分组。

D.1 脚手架构造件

名称 类型 描述 承接的结构
Vite 构建工具 + 脚手架 快速启动、HMR、构建优化 工程壳结构
Next.js 全栈框架 React 全栈、SSR、SSG 工程壳 + 页面结构
Nuxt 全栈框架 Vue 全栈、SSR、SSG 工程壳 + 页面结构
create-vite 脚手架 CLI 交互式项目创建 工程壳结构
create-react-app 脚手架 CLI React 项目创建 工程壳结构

D.2 UI 框架构造件

名称 类型 描述 承接的结构
React UI 框架 组件化、声明式、虚拟 DOM 组件组合结构
Vue UI 框架 渐进式、模板语法、响应式 组件组合结构
Svelte UI 框架 编译式、零运行时 组件组合结构
Solid UI 框架 细粒度响应式 组件组合结构

D.3 组件库构造件

名称 类型 描述 承接的结构
MUI (Material UI) 组件库 Material Design 组件 UI 展示结构
Ant Design 组件库 企业级中后台组件 UI 展示结构
shadcn/ui 组件库 可定制、Tailwind CSS 组件 UI 展示结构
Chakra UI 组件库 可访问性优先的组件 UI 展示结构
Tailwind CSS 样式框架 原子化 CSS 样式结构
Radix UI 无样式组件库 可访问性、无样式 组件行为结构

D.4 路由构造件

名称 类型 描述 承接的结构
React Router 路由库 声明式路由、嵌套路由 页面流转结构
Vue Router 路由库 Vue 官方路由 页面流转结构
TanStack Router 路由库 类型安全、懒加载 页面流转结构
Next.js Router 框架内置 文件系统路由 页面流转结构

D.5 全局状态构造件

名称 类型 描述 承接的结构
Redux Toolkit 状态库 可预测状态容器 全局状态结构
Zustand 状态库 轻量、无 boilerplate 全局状态结构
Pinia 状态库 Vue 官方状态管理 全局状态结构
Jotai 状态库 原子化状态 全局状态结构
Valohai 状态库 基于 signal 的状态 全局状态结构

D.6 服务端数据状态构造件

名称 类型 描述 承接的结构
TanStack Query 数据请求库 缓存、自动刷新、loading 服务端数据状态
SWR 数据请求库 优先缓存、stale-while-revalidate 服务端数据状态
Apollo Client GraphQL 客户端 GraphQL 数据管理 服务端数据状态
RTK Query 数据请求库 Redux Toolkit 内置 服务端数据状态

D.7 表单构造件

名称 类型 描述 承接的结构
React Hook Form 表单库 减少 re-render、验证集成 输入交互结构
Formik 表单库 表单状态管理 输入交互结构
Vue Form 表单库 Vue 表单管理 输入交互结构

D.8 验证构造件

名称 类型 描述 承接的结构
Zod 验证库 TypeScript 优先、类型推导 字段约束结构
Yup 验证库 链式 API、异步验证 字段约束结构
Valibot 验证库 模块化、树摇优化 字段约束结构
Joi 验证库 丰富的验证规则 字段约束结构

D.9 反馈构造件

名称 类型 描述 承接的结构
notistack 通知库 堆叠通知、Snackbar 反馈结构
react-hot-toast 通知库 轻量 Toast 反馈结构
sonner 通知库 简洁 Toast 通知 反馈结构
SweetAlert2 弹窗库 美观确认弹窗 反馈结构

D.10 可视化构造件

名称 类型 描述 承接的结构
ECharts 图表库 丰富图表类型、大数据量 可视化输出结构
Recharts 图表库 React 原生、声明式 可视化输出结构
D3.js 可视化库 底层灵活、自定义图表 可视化输出结构
Chart.js 图表库 简单易用、轻量 可视化输出结构

D.11 测试构造件

名称 类型 描述 承接的结构
Vitest 测试框架 原生 Vite、兼容 Jest API 测试验证结构
Jest 测试框架 零配置、Snapshot 测试验证结构
Testing Library 测试工具 用户行为测试 测试验证结构
Playwright E2E 框架 跨浏览器、自动等待 E2E 测试结构
Cypress E2E 框架 时间旅行、实时重载 E2E 测试结构
Storybook 组件开发 组件隔离开发、文档 组件文档结构
